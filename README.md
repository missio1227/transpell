# 📝 맞춤법 교정 & 영어 번역 도구

Python과 Streamlit으로 만든 한국어 맞춤법 교정 및 영어 ↔ 한국어 자동 번역 웹 애플리케이션입니다.

---

## ✨ 주요 기능

- ✅ **맞춤법 교정**: py-hanspell 기반 맞춤법 자동 수정
- 🌐 **영어 ↔ 한국어 번역**: DeepL API를 통한 고품질 번역
- 🖥️ **실시간 결과 표시**: 교정 및 번역 결과를 실시간으로 UI에 표시
- 📐 **형식 유지**: 줄바꿈 및 공백 포맷 유지
